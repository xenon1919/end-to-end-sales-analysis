# End-to-End Sales Analysis

## Overview

It is an end-to-end data analytics
project was built using Python. It demonstrates a complete data pipeline
starting from data generation and cleaning to exploratory data analysis,
feature engineering, web scraping, and finally delivering insights
through a Flask-based web dashboard.

This project focuses on transforming raw and external data into
meaningful, business-ready insights.

------------------------------------------------------------------------

## Key Features

-   Synthetic but realistic sales dataset (1000+ records)
-   Data cleaning and preprocessing using Pandas
-   Exploratory Data Analysis (EDA) with visualizations
-   Feature engineering for business insights
-   Web scraping to enrich internal data with external market data
-   Flask-based dashboard displaying KPIs and insights
-   Clear project structure suitable for real-world use

------------------------------------------------------------------------

## Tech Stack

-   **Python**
-   **Pandas, NumPy**
-   **Matplotlib, Seaborn**
-   **BeautifulSoup, Requests**
-   **Flask**
-   **HTML/CSS**

------------------------------------------------------------------------


## Dashboard Insights

-   Total Revenue, Total Orders, Average Order Value
-   Top-performing product category
-   Best-performing region
-   Discount impact analysis
-   Market price comparison using scraped data
-   Revenue distribution visualizations

------------------------------------------------------------------------

## How to Run the Project

### 1. Install Dependencies

``` bash
pip install -r requirements.txt
```

### 2. Run the Flask App

``` bash
python app/app.py
```

### 3. Open in Browser

    http://127.0.0.1:5000
